# VRML_PcRoom
VRMLで大学の演習室を再現したものです。

# Requirement
* Internet Explorer
* CORTONA3D VIEWER

# Installation

http://www.cortona3d.com/en/cortona3d-viewer-download

# Usage
* "PCroom.wrl"をダブルクリックすると実行できます。
* 事前に"CORTONA3D VIEWER"をダウンロードしてください。
* 実行時には"Internet explorer"を選択してください。
